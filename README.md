#  Análisis Exploratorio y Financiero de Registros Clínicos

##  Objetivo
Transformar datos crudos de registros hospitalarios en información accionable para la toma de decisiones, optimizando la visión operativa y financiera del centro de salud.

##  Hallazgos Clave 
* **Patología más frecuente:** Neumonía Lobular
* **Gasto Principal:** El diagnóstico de [Neumonía Lobular] representa el [42.066421]% del costo total.
* **Pico de Demanda:** Se detectó mayor volumen de ingresos en el mes de [October].

##  Tecnologías
* Python (Pandas, NumPy)
* Análisis de series temporales y limpieza de datos (Data Cleaning).
